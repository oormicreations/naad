Variations and Effects  

Naad provides controls for varying the Tempo, Bols, and Randomness etc. These enhance the creativity and usefulness of the program.

 

 

Composition Tools

1. Auto Compose will compose an Indian classical solo of Tabla in a single click. This tool is under construction and at this time it only creates some empty loops as placeholders.

 

 

Tempo Control

This slider increments or decrements the tempo or Gati. The Gati is actually number of Maatras per minute. Do not confuse it with number of Bols per minute. If there are many Bols in the Maatra they are speeded up or slowed down so that the Maatra completes in specified time.

The changes are applied to the Loop Tempo as specified in the Loop properties. The changes are shown as Deviation below the slider, along with the original tempo and the resulting tempo.

Vilambit is slow, Madhya is medium and Drut is fast. The button marked 1 X cycles through 1X, 2X, 3X, 4X, 1/4X, 1/3X, 1/2X and 1X of the tempo on each click. This is used to quickly set the tempo to integral multiples or divisions of the original.

Changes in tempo using Tempo Control are temporary and are not saved with the composition. To make the tempo setting permanent, edit the Loop properties.

 

 

Accompaniments

Accompaniments are instruments played along with the composition. You can either specify an accompaniment for the Loop in Loop properties or for the entire composition.

Checking the Ghungru at Sam checkbox will enable playing Ghungru right at the Bol marked as Sam. This is useful for keeping track of the Taal. A few varieties of Ghungrus are available from menu Accompaniment -> Ghungru.

 

 

Timing Variation and Randomness

This slider, if set at a value other than zero, will introduce a slight random variation in the tempo. This makes the Tabla a bit more natural to ears. It goes from Too Perfect for a master player to Imperfect for a novice player effect.

 

 

Bol Variety

Naad can introduce a variation in Bols during play in real time, just like a Tabla player improvising on his/her composition during a performance. This gives rise to interesting Variants (Prakar) just by sliding this slider.

The variant status is shown on the grid header.

  1. The slider introduces a new variation on each adjustment.
    • At zero position the original is played.
    • As you go from Narrow to Medium to Wide, the deviation from the original increases.
    • To play the original, click the Play button anytime.
    • Works only while playing.
  2. Checking this box will affect the entire composition and all Loops are varied.
  3. Checking this box will force the varient to be restricted within the Bol species. So a Bol Na01 will be changed to anything from Na01 to Na10, but not to any other species.
  4. Checking this box will force the variant to be restricted to the currently opened loop. The rest of the composition is unaffected.
  5. Checking this box will result in a varient with the Bols totally changed to any species, completely changing the Loop.
  6. Click this button to replace the original with the variant, if you find the variant interesting. The change is permanent after clicking this button and the original is lost. So make sure you save your composition before clicking this button.
