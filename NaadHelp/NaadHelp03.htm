Editing

Naad supports selection, cut, copy, paste and deletion of a group of Bols or Loops. This makes compositing easier and faster.

By default the views are set to Compose Mode. To enter the Edit Mode, click on menu Edit->Select. Now you can do the edit operations. To end the edit mode click Edit->Select again.

The current mode is displayed on the grid header.

Editing in Loop View

  • Enter the edit mode by clicking Edit->Select on the menu.
  • Click on the boxes one by one, as many as you want. They will be highlighted with orange color.
  • Click Edit->Copy to copy the boxes.
  • Select an empty box and click Edit->Paste to paste the boxes.
    • The relative positions of the boxes are maintained.
    • Pasting on an existing box will overwrite it.
    • Paste as many times as you want.
    • Paste also works in Compose Mode.
    • To paste the Bols of one loop into another loop, copy the boxes, end the edit mode, switch to composition mode, open another loop, select a box and paste there.
    • Pasting the Bols in Composition View will create a new loop with the copied contents.
  • To move the contents, use the Edit->Cut and Edit->Paste commands.
  • To delete the selected Bols, use Edit->Clear.
  • To end the edit mode click Edit->Select again.

 

Editing in Composition View.

The procedure is exactly like that of Loop View. You can copy-paste many loops at a time. Pasting a Loop in Loop View will result in pasting of its all Bols there. (If many Loops were copied only the Bols of first Loop shall be pasted in this case.)

As in case of Bols, the relative positions of the Loops are maintained. Ensure that you select a location for pasting which provides enough room so that no Loops are overwritten.

 

Adding of Bols to existing Bols

In order to simulate hitting of both the drums at the same time, one can add an extra bol to the existing one.

This also gives the effect of two Tablas playing at the same time, if all bols are appended with an extra and a different varient is used for the extra.

To add a bol to the existing one, click Edit->Add and click the bol to set additional bol at the same place.

Click Edit->Add again to come back to compose mode.

There is no Undo command at this time.

Please save your composition before doing a big change. You can trash the changes if something goes wrong and re-load the saved file.
