Composition Structure

A Composition contains Loops and loops contain the Bols (Recorded Samples). To compose, you have to create one or more Loops and create one or more Bols in each Loop. When you click the Play button, the Bols in the Loops are played in a sequence and the loops are also played in a sequence.

Naad presents two kinds of views on the same grid:

Composition View : Here you can see the layout of the Loops, their names and other info. Each loop is shown in a box. The sequence of playing the loops is from left to right and top to bottom. Any blank boxes are ignored, so you can put your loops anywhere you want. This view opens by default when you click on the menu File->New Composition.

Loop View : Click on a loop and click Open on the context menu which pops up, you are now in the Loop View. Here you can see the Bols. To go back to Composition View, click anywhere on the grid and click View Composition from the menu.

 

Bols

A Bol is the basic sound produced by Tabla (or any other Indian percussion instrument), when struck in a particular way. Tabla can produce hundreds of different Bols, depending on the drum and where it is struck. Naad uses 120 varieties of Bols, categorized in 12 species. This means there are 10 varieties of Bols for each species. The species are shown below. (These are nothing but the traditional names of the Bols).

  1. Na
  2. Ta
  3. Ti
  4. Tin
  5. Tun
  6. Dha
  7. Dhin
  8. Dhit
  9. Dhun
  10. Ga
  11. Ge
  12. Ke

Out of these 1-5 are produced on the small drum of Tabla (Dayan), 10-12 are produced on the big drum or the bass drum (Bayan) and 6-9 are produced by simultaneous hits of Dayan and Bayan. The variations of the Na species are named as Na01, Na02...Na10, and so on. Each variation is a wav file having actual recorded Bol of that type. Naad does not use any synthetic sounds, you hear the pure authentic Tabla. You can find these wav samples in the folder named Bol. Naad loads these samples from here.

Specifying a Bol in your composition

  • On menu click File->New Composition
  • Click on any of the boxes in the grid and from the context menu select Open
  • Click on any of the boxes in the grid and hover on either Dayan or Bayan items, which will pop up another sub-menu showing the species. Hovering on these will pop up the variations of the species. Hovering on the variation produces the sound of the Bol (Audible preview). Select a Bol by clicking on it.
  • The box will show the selected bol. Dayan Bols are shown in Red and the rest are shown in Blue.
  • Click the Play Loop button to test.

The Bol Folder

This folder has the wav samples, which are used to render your composition. However, you can specify your own samples simply by changing to another folder where your own samples are located by clicking on menu Options->Bol Folder.

Note that your folder can be of any name but it must have the same file names and subfolder names as the original. The wav files can have different sounds of Tabla or even of different instrument. The effect of changing the Bol folder to your own will be that your composition will be rendered with your samples.

You can hear all the samples in Bol folder by clicking on menu Options->Play All Samples

 

 

Loops

A Loop is a string of Bols. In a composition, the loop can be repeated many times or plays only once (e.g. at the start or end or during a change of beat, but it will be still called as a Loop).

In a loop, the Bols are grouped together in a Maatra. The whole Maatra is counted as a single beat. So if a maatra contains 4 Bols, all the 4 Bols shall be played in sequence to complete that beat. Of course, the Maatra can have a single Bol, in which case the number of Maatras in that loop is equal to the number of Bols.

A Maatra can also have nothing (Silence ~o~). So the silence is also treated as a Bol.

Creating a Loop

  • On menu click File->New Composition
  • Click on any of the boxes in the grid and from the context menu select Open. This opens the Loop for editing.
  • Specify the Bols in the loop, one Bol per box. (How?)
  • Naad ignores the blank boxes while playing, so you can put your Bols anywhere, however keep in mind that the playing sequence is from left to right and top to bottom.
  • Click the Play Loop button to test.

Context Menu

In the Loop View you will see this context menu whenever you click on a box.

  1. Switches back to Composition View
  2. Select the Dayan Bols
  3. Select the Bayan Bols
  4. Select a commonly used piece
  5. Insert Silence (Note that a blank box is not silence, it will be skipped during play and the next Bol is played immediately. When you select Silence, there is a gap in play)
  6. Specifies if the Bol is a Sam (Applies to an existing Bol. No effect on play, except when Ghungru on Sam check box is checked on the main UI, whence you hear a ghungru whenever the Sam Bol plays. This is used to keep track of the beat during a dance practice, for example.)
  7. Specifies if the Bol is Tali (Applies to an existing Bol. No effect on play.)
  8. Specifies if the Bol is Khali (Applies to an existing Bol. No effect on play.)
  9. Clears the box

Loop Properties

The loop may be repeated few times in a composition, or it may have many Bols in a Maatra. You can set these behaviours and properties via the Loop Properties Dialog. Click on menu View->Loop Properties to see this dialog.

  1. Name of the Loop. Same will appear in composition view, in its box there.
  2. Number of Bols per Maatra. The Grid view is shaded based on how many bols are there in one Maatra.
  3. Number of Maatras in the Loop are auto-computed based on how many Bols are entered.
  4. Number of times this loop will be repeated in a composition.
  5. Tempo in Beats per minute. Each loop can have its own Tempo. (Note that in Naad the Tempo is actually Maatra per minute.)
  6. Any accompaniment to be played with the loop can be selected here.
  7. Notes for composer's own purpose.

 

Compositions  

A Composition is a string of Loops. These loops are shown and created on the grid, which is called the Composition View. Click anywhere in this view to pop up the context menu for composition.

  1. Opens the Loop for editing and also creates it.
  2. Opens the Loop Properties Dialog.
  3. Loads a previously saved Loop. (A Naad Loop File having extension.nlp)
  4. Saves the clicked loop as .nlp file.
  5. Clears the box and deletes the Loop.

You can load a Naad version 0.2 file (.nad file) as a loop.

Click on the Load Loop and in the file selection dialog set the filter to All Files. Select the Naad v 0.2 file and click Open. This file will be converted to current version and will be loaded as a loop.

Composition Properties

On menu Click View->Composition Properties. This dialog shows the name of the composition, composer and any notes etc. Edit this to your likeness. When editing or improvising someone else's composition, be sure to keep the original names and notes as they are and append yours to it.
